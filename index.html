Responsive Web Design

Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones) based on screen size, platform and orientation. Responsive Web Design makes use of flexible layouts, flexible images and cascading style sheet media queries.

I most frequently set major and minor width breakpoints based on viewport tag and CSS media queries, and then code to present optimized content layout and text based on the display size ranges between these breakpoints which enables content to scale in size based on the device and platform the reader of the web page is using at the time.

Non-responsive web page is one that reads well on desktop browsers but has very small, unreadable text on smartphones, often due to having too many columns or images that are too large to fit within a smartphone's limited viewport display width.But responsive design adapt to a range of display sizes, presenting different layouts based on the display size and capabilities of the mobile device or PC.
